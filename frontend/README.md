# Frontend - Hello World CI/CD Dashboard

## About

This is a simple, interactive frontend that explains and visualizes the CI/CD pipeline workflow.

## Files

- **index.html** - Interactive dashboard with pipeline information
- **style.css** - Modern dark theme styling
- **script.js** - Dynamic updates and animations

## How to Use

### **Local Development**

1. **Open in Browser:**
   - Double-click `index.html` file
   - Or use Live Server in VS Code

2. **View Pipeline Information:**
   - See pipeline flow diagram
   - Learn how to check status
   - Find quick commands

3. **Open in Live Server (VS Code):**
   - Install "Live Server" extension
   - Right-click on `index.html`
   - Select "Open with Live Server"
   - Browser opens at `http://localhost:5500`

### **Access Remotely**

Once deployed, access at your deployment URL.

## Understanding Pipeline Flow

The frontend shows 5 main steps:

1. **Code Push** → Push code to repository
2. **Checkout** → Clone code into virtual machine
3. **Build** → Run build commands (print Hello World)
4. **Test** → Execute tests
5. **Deploy** → Final verification and success

## Key Information Displayed

- **Pipeline Flow Diagram** - Visual representation of workflow
- **How to Check Status** - Navigate to GitHub Actions
- **How Pipeline Works** - Technical explanation
- **Quick Commands** - Terminal commands to trigger pipeline
